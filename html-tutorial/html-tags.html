Session - 1: Introduction to Client-side and HTML

HTML Elements, Tags and Attributes #

  • HTML is a markup language making use of series of elements, which is basically use to wrap, enclose the content to render it on the screen, paper or any media.
  • HTML elements are creating using tags.
  • Elements can have various different attributes providing additional information about elements. It is always defined in the starting tag.
  • Anatomy of an Element:
  • Elements can be nested in one another.
Type Tag Description
Root html Represents root of an HTML document, referred as Root Elements.
Meta head contains meta data about document
meta represents metadata
link specifies relationship between current document and external resource, for loading CSS files, favicon, etc.
style contains style infomation related to document
title defines document title
Sectioning Root body represents content of an HTML Document.
Text content div division element defines a section in a document
hr defines thematic break in HTML Page (horizontal rule)
ul defines unordered list
ol defines ordered list
li defines list item
pre represents preformatted text which renders exactly as written within the tag
Inline Text tags a anchor element, creates hyperlink to web pages, files, email, phone with its "href" attribute
b defines bring attention to element
br produces link break in text
span generic inline container, specifies section in document
strong defines strong importance of content
small used for printing small texts, by default renders one font size smaller.
Media img Embeds image into the document
video Embeds a media player into the document
map used with area elements, to define an image map
Scripting script Used to embed executable code or data
canvas use to draw graphics and animations like charts
Other iframe embeds another HTML page into current one

Few HTML elements have no content. These kind of elements are called Empty elements.
